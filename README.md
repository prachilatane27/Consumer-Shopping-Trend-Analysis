# Consumer Shopping Trend Analysis

## Project Overview
This project analyzes consumer shopping behavior using Python, SQL, and Power BI. The objective is to identify customer spending patterns, shopping preferences, and business opportunities.

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- SQL
- Power BI
- Google Colab

## Dataset Features
- Age
- Monthly Income
- Shopping Preference
- Avg Online Spend
- Avg Store Spend
- Social Media Hours
- Impulse Buying Score
- Brand Loyalty Score
- City Tier

## Key Analysis Performed
- Data Cleaning & Preparation
- SQL Business Queries
- Customer Behavior Analysis
- Spending Pattern Analysis
- Dashboard Creation

## Key Insights
- Online shoppers contribute higher revenue.
- High-income customers spend more.
- Tier 3 cities show strong spending growth.
- Social media usage influences impulse buying.
